# Harness Design for Long-Running Application Development

> 来源：[Anthropic Engineering Blog](https://www.anthropic.com/engineering/harness-design-long-running-apps)
> 作者：Prithvi Rajasekaran（Anthropic Labs）
> 发布日期：2026 年 3 月 24 日

## 文章概述

本文探讨了**多智能体 Harness（编排框架）设计**如何显著提升 Claude 在复杂、长时间运行任务上的表现，尤其是前端设计和全栈应用开发场景。

## 核心问题

1. **自我评估失效**：模型评估自己的工作时倾向于过度正面评价，即使输出质量一般。这在设计等主观任务中尤为突出。
2. **上下文管理焦虑**：模型在接近上下文窗口限制时会"焦虑"地草草收尾。研究发现，上下文重置（而非摘要）能提供干净的工作环境，维持连贯性。

## 核心创新：生成器-评估器架构（Generator-Evaluator）

受 GAN（生成对抗网络）启发，将**工作生成**与**工作评估**分离为不同智能体：

- 调优一个"怀疑论者"评估器，比让生成器自我批评更有效
- 具体反馈循环驱动迭代改进
- 关注点清晰分离

## 前端设计应用

定义了四个评分标准，将主观的美学判断转化为可衡量的维度：

| 维度                                 | 说明                             |
| ------------------------------------ | -------------------------------- |
| **设计质量（Design Quality）** | 颜色、排版、布局的视觉一致性     |
| **原创性（Originality）**      | 自定义创意 vs 千篇一律的 AI 风格 |
| **工艺（Craft）**              | 层级、间距、配色等技术执行力     |
| **功能性（Functionality）**    | 可用性与任务完成度               |

评估器使用 **Playwright** 与实际实现交互，截图并分析设计后提供详细反馈。通常经过 **5-15 轮迭代**，推动生成器产出更具辨识度的作品。

## 全栈开发 Harness

采用**三智能体系统**：

1. **Planner Agent（规划智能体）**：将简短提示（1-4 句话）扩展为完整的产品规格文档，融入 AI 功能，避免过度指定实现细节。
2. **Generator Agent（生成智能体）**：以 Sprint 方式实现功能，使用 React、Vite、FastAPI、SQLite/PostgreSQL 技术栈，配合 Git 版本控制。
3. **Evaluator Agent（评估智能体）**：使用 Playwright 像用户一样测试应用，检查 UI 功能、API 和数据库状态，对照预先协商的 Sprint 合约验收。

## 效果对比

以"复古游戏制作器"提示为例：

| 方式         | 耗时    | 成本 |
| ------------ | ------- | ---- |
| 单智能体     | 20 分钟 | $9   |
| 完整 Harness | 6 小时  | $200 |

单智能体产出的应用存在严重缺陷（实体不响应输入），而 Harness 产出了功能完整的游戏体验，包括精灵编辑器和 AI 辅助关卡设计。

## 随模型进化简化 Harness

当 Claude Opus 4.6 发布后（具备更好的规划、更长任务持续力、更强调试能力），研究者**系统性地移除 Harness 组件**：

- Sprint 结构被证明可以移除，性能不受损失
- 说明对模型局限性的假设应**持续验证**

典型案例：一个数字音频工作站（DAW）构建仅用 3 小时 50 分钟，模型连续工作 2 小时以上无需 Sprint 分解。

## 关键洞察

1. **分离生成与评估**比自我批评更易调优
2. **Harness 复杂度应随模型进步而降低**——不应视为永久脚手架
3. **评分标准和提示词**直接塑造输出风格，甚至在迭代反馈之前就已起效
4. **基于文件的结构化交接**能在长会话中有效保留上下文

## 核心结论

随着 Claude 能力的提升，AI 工程的有趣空间在**迁移而非缩小**。实践建议：

- 持续审视哪些 Harness 组件仍是承重结构
- 剥离不必要的复杂性
- 探索利用新能力的创新组合
